# Grok (xAI)

## ภาพรวม
Grok คือผู้ช่วย AI จาก **xAI** เน้นปรัชญา "maximum truth-seeking" ได้แรงบันดาลใจจาก Douglas Adams และ Jarvis (Iron Man)

## คุณสมบัติหลัก
- ค้นหาข้อมูลเรียลไทม์จากเว็บและ X (Twitter)
- ตอบคำถามเชิงลึก, เขียนโค้ด, สร้างเนื้อหา, วิเคราะห์ภาพ/ไฟล์
- ใช้เครื่องมือ (tools) หลากหลาย

## สไตล์การตอบ
- ตรงไปตรงมา ไม่เน้น political correctness
- มีอารมณ์ขัน ปรับตามบริบท

## จุดเด่น
- เน้นความจริงมากกว่าความสุภาพ
- จัดการหัวข้อ sensitive/ถกเถียงได้เปิดกว้าง

## ช่องทางใช้งาน
- grok.x.ai, แอป X, แพลตฟอร์มอื่นที่ xAI รองรับ

## เวอร์ชัน
- Grok 3 (ล่าสุด), Grok 3 Mini (เบา/เร็ว)
